# hubot-shorts

See [`src/shorts.coffee`](src/shorts.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-shorts --save`

Then add **hubot-shorts** to your `external-scripts.json`:

```json
[
  "hubot-shorts"
]
```
